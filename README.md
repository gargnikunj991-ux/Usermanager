# Library Management System v1.0

A console-based Library Management System built using Java and Object-Oriented Programming principles. The project allows library staff to manage books, members, and employees while supporting authentication, borrowing, returning, and file-based data persistence.

## Features

### Authentication

* User Login
* User Logout
* Session Management
* Login Status Check

### Employee Management

* Add User
* Search User
* Update User
* Delete User
* Change Username
* Display Users

### Member Management

* Add Member
* Search Member
* Display Members
* File Persistence

### Book Management

* Add Book
* Search Book
* Display Books
* File Persistence

### Library Operations

* Borrow Book
* Return Book
* Track Borrowed By
* Track Issued By

## Technologies Used

* Java
* Object-Oriented Programming (OOP)
* HashMap Collections
* File Handling (FileReader, FileWriter, BufferedReader)
* Git & GitHub

## How to Run

1. Clone the repository

```bash
git clone <repository-url>
```

2. Open the project in VS Code or any Java IDE

3. Compile and run:

```bash
javac *.java
java Main
```
